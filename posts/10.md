## Fork repository

```jsx
git fetch upstream
git merge upstream/main
git push origin main
git switch 작업 브랜치
git rebase main
# conflict 수정후
git rebase --continue
git push origin 작업브랜치 -f
```

## clone mirror

```
git clone --mirror {old_url}
cd {project_name}.git
git remote set-url --push origin {new_url}
git push --mirror
```
